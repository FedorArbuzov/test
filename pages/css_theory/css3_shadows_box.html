CSS - Тень коробки

CSS Свойство box-shadow

CSS Свойство box-shadow применяет тень к элементам.

В простейшем использовании вы указываете только горизонтальную тень и вертикальную тень:

Это желтый элемент <div> с черной тенью

Пример

div {
  box-shadow: 10px 10px;
}

Далее, добавить цвет в тень:

Это желтый элемент <div> с серой тенью

Пример

div {
  box-shadow: 10px 10px grey;
}

Далее, добавьте эффект размытия к тени:

Это желтый элемент <div> с размытой серой тенью

Пример

div {
  box-shadow: 10px 10px 5px grey;
}

Карточки

Пример

div.card {

  box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
  text-align: center;
}

Проверьте себя с помощью упражнений!
