CSS - border-radius

CSS border-radius

Свойство border-radius используется для добавления округлых границ к элементу:

Нормальная граница

Круглая граница

Более круглая граница

Самая круглая граница

Пример

p {
  border: 2px solid red;
  border-radius: 5px;
}
