CSS - Размер таблицы (ширина и высота)

Ширина и высота таблицы

Ширина и высота таблицы определяются свойствами width и height.

В приведенном ниже примере ширина таблицы устанавливается равной 100%, а высота элементов <th> - 70px:

Пример

table {
  width: 100%;
}

th {
  height: 70px;
}

Чтобы создать таблицу, которая должна занимать только половину страницы, используйте width: 50%:

Пример

table {
  width: 50%;
}

th {
  height: 70px;
}
