CSS - outline

CSS outline

Свойство outline - это сокращенное свойство для настройки следующих отдельных свойств контура:

  • outline-width
  • outline-style (требуемый)
  • outline-color

Свойство outline указывается как одно, два или три значения из приведенного выше списка. Порядок значений не имеет значения.

В следующем примере показаны некоторые контуры, заданные с помощью сокращенного свойства outline:

Пунктирный контур.

Пунктирный красный контур.

Сплошной желтый контур размером 5 пикселей.

Толстый розовый контур хребта.

Пример

p.ex1 {outline: dashed;}
p.ex2 {outline: dotted red;}
p.ex3 {outline: 5px solid yellow;}
p.ex4 {outline: thick ridge pink;}
