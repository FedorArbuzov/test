CSS - Сокращенное свойство Font

CSS Свойство Font

Чтобы сократить код, также можно указать все отдельные свойства шрифта в одном свойстве.

Свойство font является сокращенным свойством для:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Примечание: Значения font-size и font-family обязательны. Если одно из других значений отсутствует, используется их значение по умолчанию.

Пример

Используется font для задания нескольких свойств шрифта в одном объявлении:

p.a {
  font: 20px Arial, sans-serif;
}

p.b {
  font: italic small-caps bold 12px/30px Georgia, serif;
}

Проверьте себя с помощью упражнений!
