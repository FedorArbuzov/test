CSS border - сокращенное свойство

Как вы видели на предыдущей странице, есть много свойств, которые следует учитывать при работе с границами.

Чтобы сократить код, можно также указать все отдельные свойства границы в одном свойстве.

Свойство border - это сокращенное свойство для следующих отдельных свойств границ:

  • border-width
  • border-style (требуемый)
  • border-color

Пример

p {
  border: 5px solid red;
}

Вы также можете указать все отдельные свойства границ только для одной стороны:

Граница левая

p {
  border-left: 6px solid red;
  background-color: lightgrey;
}
