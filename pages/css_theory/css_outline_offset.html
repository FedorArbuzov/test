CSS outline-offset

Свойство outline-offset добавляет пробел между контуром и краем/границей элемента. Пространство между элементом и его контуром является прозрачным.

В следующем примере задается контур размером 15 пикселей за пределами края границы:

Этот параграф имеет контур размером 15 пикселей за пределами границы.

Пример

p {
  margin: 30px;
  border: 1px solid black;
  outline: 1px solid red;
  outline-offset: 15px;
}

В следующем примере показано, что пространство между элементом и его контуром является прозрачным:

Этот параграф имеет контур размером 15 пикселей за пределами границы.

Пример

p {
  margin: 30px;
  background: yellow;
  border: 1px solid black;
  outline: 1px solid red;
  outline-offset: 15px;
}

Проверьте себя с помощью упражнений

Упражнение:

Установите сплошную границу контура размером 5 пикселей для элемента <div>.

<style>
div {
: solid;
: 5px;
}
</style>
