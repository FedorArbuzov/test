CSS - Псевдо-классы

Что такое псевдо-классы?

Псевдо-класс используется для определения особого состояния элемента.

Например, он может быть использован для:

  • Стиля элемента при наведении курсора мыши
  • Стиля посещенных и непосещенных ссылок
  • Стиля элемента, для получения фокуса

Наведении курсора мыши

Синтаксис

Синтаксис псевдо-класса:

Селектор: псевдо-класс {
   Свойство: значение;
}

Установить псевдо-класс

Ссылки могут отображаться различными способами:

Пример

/* непросмотренная ссылка */
a:link {
   color: #FF0000;
}

/* просмотренная ссылка */
a:visited {
   color: #00FF00;
}

/* наведение мыши на ссылку */
a:hover {
   color: #FF00FF;
}

/* активная ссылка */
a:active {
   color: #0000FF;
}

Примечание: a:hover должен перейти после a:link и a:visited в определение CSS для того, чтобы быть эффективней! a:active должен перейти после a:hover в определение CSS для того, чтобы быть эффективнее! Псевдо-классы не чувствительны к регистру.

Псевдо-классы и классы CSS

Псевдо-классы должны быть объединены с классами CSS:

При наведении курсора на ссылку в примере ниже, она изменит цвет:

Пример

a.highlight:hover {
   color: #ff0000;
}

Наведите на <div>

Пример использования псевдо-класса :hover элемента <div>:

Пример

div:hover {
   background-color: blue;
}

Наведите на <p>

Наведите курсор на элемент <div> для элемента <p> (как всплывающая подсказка):

<p>Наведите курсор на меня, чтобы показать элемент </p>

Подсказка!

Пример

p {
   display: none;
   background-color: yellow;
   padding: 20px;
}

div:hover p {
   display: block;
}

Псевдо-класс - :first-child

Псевдо-класс :first-child соответствует заданному элементу, который является первым ребенком элемента, другого элемента.

Первый ребенок параграфа

В следующем примере селектор сопоставляется каждому элементу <p>, который является первым ребенком элемента:

Пример

p:first-child{
   color: blue;
}

Первый ребенок элемента <i> во всех <p>

В следующем примере селектор сопоставляется первый элемент <i> во всех элементах <p>:

Пример

p i:first-child{
   color: blue;
}

Все дети элементов <i> в первом элементе <p>

В следующем примере селектор сопоставляется всем элементам <i> в элементе <p>, что первый ребенок от другого элемента:

Пример

p:first-child i{
   color: blue;
}

Псевдо-класс - :lang

Псевдо-класс :lang позволяет определить специальные правила для разных языков.

В пример ниже, :lang определяет кавычки для элементах <q> с lang="no":

Пример

<html>
<head>
<style>
q:lang(no){
   quotes: "~" "~";
}
</style>
</head>

<body>
<p>Начало текста <q lang="no">Цитата в параграфе</q> конец текста.</p>
</body>
</html>

Все CSS псевдо-классы

Селектор Пример Описание
a:active Устанавливает активную ссылку
input:checked Устанавливает каждый элемент проверки в <input>
input:disabled Устанавливает каждый элемент отключение в <input>
p:empty Устанавливает каждый элемент <p>, который не имеет детей
input:enabled Устанавливает каждый элемент включение в <input>
p:first-child Устанавливает каждый элемент <p>, является первым ребенком своего родителя
p:first-of-type Устанавливает каждый элемент <p>, является первым элементом <p> своего родителя
input:focus Устанавливает элемент <input>, который имеет фокус
a:hover Выбирает ссылку при наведение курсором
input:in-range Выбирает элементы <input> в пределах указанного диапазона значений
input:invalid Выбирает все элементы <input> с недопустимым значением
p:lang(it) Выбирает каждый элемент <p> со значением атрибута lang, начиная с "it"
p:last-child Выбирает каждый элемент <p> что является последним ребенком своего родителя
Выбирает каждый элемент <p>, который является последним элементом <p> своего родителя
a:link Выбирает все непосещенные ссылки
:not(p) Выбирает каждый элемент которого нет в элементе <p>
p:nth-child(2) Выбирает каждый элемент <p>, что это второй ребенок своего родителя
p:nth-last-child(2) Выбирает каждый элемент <p>, который является вторым ребенком своего родителя, считая от последнего ребенка
p:nth-last-of-type(2) Выбирает каждый элемент <p> вторичного элемента <p> своего родителя, считая от последнего ребенка
p:nth-of-type(2) Выбирает каждый элемент <p> вторичного элемента <p> своего родителя
p:only-of-type Выбирает каждый элемент <p> , который является единственным элементом <p> своего родителя
p:only-child Выбирает каждый элемент <p>, который является единственным ребенком своего родителя
input:optional Выбирает элемент <input> без атрибута "required"
input:out-of-range Выбирает элемент <input> со значением, выходящим за пределы указанного диапазона
input:read-only Выбирает элемент <input> с определенным атрибутом "readonly"
input:read-write Выбирает элемент <input> без атрибута "readonly"
input:required Выбирает элемент <input> с определённым атрибутом "required"
root Выбирает корневой элемент документа
#news:target Выбирает текущий активный #news элемент (нажали на URL-адрес, содержащий имя якоря)
input:valid Выбирает все элементы <input> с допустимым значением
a:visited Выбирает все посещенные ссылки
