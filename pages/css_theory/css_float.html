CSS - float

Cвойство float указывает, является ли элемент поплавком.

Cвойство clear используется для управления поведением плавающего элемента.



Cвойство поплавок

Свойство float может использоваться, чтобы обернуть текст вокруг изображения.

В примере указывается, что изображение смещаеться в тексте вправо:

Пример

img {
   float: right;
   margin: 0 0 10px 10px;
}

Cвойство очистки

Cвойство clear используется для управления поведением плавающего элемента.

Элемент установленный после плавающего элемента будет плавать вокруг него. Чтобы избежать этого, используйте свойство очистки clear.

Свойство clear определяет, на какой стороне элемента плавающие элементы не будут плавать:

Пример

div {
   clear: left;
}

Класс .clearfix

Если элемент находится выше, чем элемент содержащий его, и изображение плавает за пределами своего контейнера.

Чтобы исправить эту проблему, мы можем добавить в класс .clearfix содержащий элемент overflow: auto;:

Пример

.clearfix {
   overflow: auto;
}

Элемент overflow: auto; в классе .clearfix хорошо работает, если Вы не в состоянии держать под контролем свои поля и отступы (можно увидеть полосы прокрутки). Класс .clearfix использовать безопасно, следующий код используется для большинства веб-страниц:

Пример

.clearfix::after {
   content: "";
   clear: both;
   display: table;
}

Вы узнаете больше о ::after псевдо-элементе в более поздней главе.

Пример веб-макета

Он является общим, чтобы сделать веб-макет с использованием свойства float:

Пример

.header, .footer {
   background-color: grey;
   color: white;
   padding: 15px;
}

.column {
   float: left;
   padding: 15px;
}

.clearfix::after {
   content: "";
   clear: both;
   display: table;
}

.menu {
   width: 25%;
}

.content {
   width: 75%;
}

Все CSS свойства поплавка

Свойство Описание
Указывает, стороны элемента, где не будут плавать элементы
Указывает, где должен плавать элемент
Указывает, что произойдет, если содержимое переполняет элемент бокса
Указывает, что делать с левым и правым краями контента, если он переполняет область содержимого элемента
Указывает, что делать с верхним и нижним краями контента, если он переполняет область содержимого элемента
