CSS - Стиль таблицы

Заполнение таблицы

Для управления пространством между границей и содержимым таблицы используйте paddingсвойство для элементов <td> and <th>:

Пример

th, td {
  padding: 15px;
  text-align: left;
}

Добавьте свойство border-bottom в <th> и <td> для горизонтальных разделителей:

Пример

th, td {
  border-bottom: 1px solid #ddd;
}

Подвесная таблица

Используйте селектор :hover в <tr> для выделения строк таблицы при наведении курсора мыши:

Фамилия Имя Экономия
Щипунов Андрей $100
Щипунова Татьяна $150
Щипунова Кристина $300

Пример

tr:hover {background-color: #f5f5f5;}

Фамилия Имя Экономия

Для полосатой таблицы используйте селектор nth-child() и добавьте background-color ко всем четным (или нечетным) строкам таблицы:

Пример

tr:nth-child(even) {background-color: #f2f2f2;}
