SQL - MIN() и MAX()

MIN() и MAX()

Функция MIN() возвращает наименьшее значение выбранного столбца.

Функция MAX() возвращает наибольшее значение выбранного столбца.

Синтаксис MIN()

SELECT MIN(column_name)
FROM table_name
WHERE condition;

Синтаксис MAX()

SELECT MAX(column_name)
FROM table_name
WHERE condition;

Пример MIN()

Следующая инструкция SQL находит цену самого дешевого продукта:

Пример

SELECT MIN(Price) AS SmallestPrice
FROM Products;

Пример MAX()

Следующая инструкция SQL находит цену самого дорогого продукта:

Пример

SELECT MAX(Price) AS LargestPrice
FROM Products;
