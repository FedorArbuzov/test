SQL - DROP TABLE

DROP TABLE

Инструкция DROP TABLE используется для откидывания существующей таблицы в базе данных.

Синтаксис

DROP TABLE table_name;

Примечание: Будьте осторожны, прежде чем откинуть таблицу. Откидывание таблицы приведет к потере полной информации, хранящейся в таблице!

Пример DROP TABLE

Следующая инструкция SQL откидывает существующую таблицу "Shippers":

Пример

DROP TABLE Shippers;

Пример TRUNCATE TABLE

Инструкция TRUNCATE TABLE используется для откидывания данных внутри таблицы, но не самой таблицы.

Синтаксис

TRUNCATE TABLE table_name;
