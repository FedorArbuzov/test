SQL - SELECT

SELECT

Инструкция SELECT используется для выбора данных из базы данных.

Возвращаемые данные хранятся в таблице результатов, называемой результирующим набором.

SELECT синтаксис

SELECT column1, column2, ...
FROM table_name;

Вот, колонка1, колонка2, ... имена полей таблицы, из которой вы хотите выбрать данные. Если вы хотите выбрать все поля, доступные в таблице, используйте следующий синтаксис:

SELECT * FROM table_name;

Выбор столбцов

Следующая инструкция SQL выбирает "CustomerName" и "Город" столбцы из таблицы "Клиенты":

Пример

SELECT CustomerName, City FROM Customers;

Выбрать все

Следующая инструкция SQL выбирает все столбцы из таблицы "Customers":

Пример

SELECT * FROM Customers;
