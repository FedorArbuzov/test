SQL - EXISTS

EXISTS

Оператор EXISTS используется для проверки наличия любой записи в подзапросе.

Оператор EXISTS возвращает true, если подзапрос возвращает одну или несколько записей.

EXISTS Syntax

SELECT column_name(s)
FROM table_name
WHERE EXISTS
(SELECT column_name FROM table_name WHERE condition);

Примеры SQL EXISTS

Следующая инструкция SQL возвращает TRUE и перечисляет поставщиков с ценой продукта менее 20:

Пример

SELECT SupplierName
FROM Suppliers
WHERE EXISTS (SELECT ProductName FROM Products WHERE Products.SupplierID = Suppliers.supplierID AND Price < 20);

Следующая инструкция SQL возвращает TRUE и перечисляет поставщиков с ценой продукта равной 22:

Пример

SELECT SupplierName
FROM Suppliers
WHERE EXISTS (SELECT ProductName FROM Products WHERE Products.SupplierID = Suppliers.supplierID AND Price = 22);
