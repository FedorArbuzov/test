SQL - JOIN

JOIN Самостоятельное соединение

Самостоятельное соединение - это обычное соединение, таблица соединяется сама с собой.

Синтаксис JOIN Самостоятельно

SELECT column_name(s)
FROM table1 T1, table1 T2
WHERE condition;

T1 и T2 это разные псевдонимы таблиц для одной и той же таблицы.

Пример SQL JOIN Сам

Следующая инструкция SQL соответствует клиентам из одного и того же города:

Пример

SELECT A.CustomerName AS CustomerName1, B.CustomerName AS CustomerName2, A.City
FROM Customers A, Customers B
WHERE A.CustomerID <> B.CustomerID
AND A.City = B.City
ORDER BY A.City;
