CSS outline-width

Свойство outline-width задает ширину контура и может иметь одно из следующих значений:

  • Тонкий (обычно 1px)
  • Средний (обычно 3px)
  • Толстый (обычно 5px)
  • Конкретный размер в ( px, pt, cm, em, и т.д.)

В следующем примере показаны некоторые контуры с разной шириной:

Пример

p.ex1 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: thin;
}

p.ex2 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: medium;
}

p.ex3 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: thick;
}

p.ex4 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: 4px;
}
