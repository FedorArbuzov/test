Использование свойства width

Если свойство width установлено на 100 процентов, видеоплеер будет отзывчивый и масштабироваться вверх и вниз:

Пример

video {
    width: 100%;
    height: auto;
}
Попробуйте сами »

Обратите внимание, что в приведенном выше примере можно увеличить размер видеопроигрывателя, чем свой первоначальный размер. Лучшим решением во многих случаях будет использование свойства max-width.

Использование свойства max-width

Если свойство max-width имеет значение 100 процентов, видео-плеер будет уменьшаться, если он должен, но никогда не масштабируйте, больше чем свой первоначальный размер:

Пример

video {
    max-width: 100%;
    height: auto;
}

Добавление видео на веб страницу пример

Мы хотим добавить видео в нашем примере веб страницы. Размер видео будет всегда занимать все доступное пространство:

Пример

video {
    width: 100%;
    height: auto;
}
