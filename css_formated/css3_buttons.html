Узнать, как стилизовать кнопки с помощью CSS3.

Основной стиль кнопок

Пример

.button {
    background-color: #4CAF50; /* Зеленый */
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
}

Цвета кнопок

Использовать свойство background-color для изменения цвета фона кнопки:

Пример

.button1 {background-color: #4CAF50;} /* Зеленый */
.button2 {background-color: #008CBA;} /* Синий */
.button3 {background-color: #f44336;} /* Красный */
.button4 {background-color: #e7e7e7; color: black;} /* Серый */
.button5 {background-color: #555555;} /* Черный */

Кнопки при наведении


Использовать селектор :hover селектор для изменения стиля кнопки при наведении мыши.

Совет: Использовать свойство transition-duration для определения скорость эффекта "наведения":

Пример

.button {
    -webkit-transition-duration: 0.4s; /* Safari */
    transition-duration: 0.4s;
}

.button:hover {
    background-color: #4CAF50; /* Зеленый */
    color: white;
}
...

Тень кнопки

Использовать свойство box-shadow для добавления теней к кнопке:

Пример

.button1 {
    box-shadow: 0 8px 16px 0 rgba(0,0,0,0.2), 0 6px 20px 0 rgba(0,0,0,0.19);
}

.button2:hover {
    box-shadow: 0 12px 16px 0 rgba(0,0,0,0.24), 0 17px 50px 0 rgba(0,0,0,0.19);
}

Отключенные кнопки

Использовать свойство opacity для добавления прозрачности к кнопке (создает "отключенный" взгляд).

Совет: Вы также можете добавить свойство cursor со значением not-allowed, который покажет "какой-то знак стоянки автомобилей" когда вы наведете мышь на кнопку:

Пример

.disabled {
    opacity: 0.6;
    cursor: not-allowed;
}

Ширина кнопки


По умолчанию размер кнопки определяется ее текстовым содержимым (шириной содержание). Использовать свойство width для изменения ширины кнопки:

Пример

.button1 {width: 250px;}
.button2 {width: 50%;}
.button3 {width: 100%;}

Группа кнопок


Удалить поля и добавить float:left для каждой кнопки, чтобы создать группу кнопок:

Пример

.button {
    float: left;
}

Границы групп кнопок


Использовать свойство border для создания границ кнопки группа:

Пример

.button {
    float: left;
    border: 1px solid green;
}

Вертикальная группа кнопок


Использовать display:block вместо float:left группировать кнопки друг под другом, а не рядом:

Пример

.button {
    display: block;
