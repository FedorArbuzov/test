CSS Комментарии

Комментарии используются для объяснения кода и могут помочь при редактировании исходного кода на более позднем этапе.

Комментарии игнорируются браузерами.

CSS комментарий помещается внутри файла элемента <style>, и начинается с /* и заканчивается */:

Пример

/* Это однострочный комментарий */
p {
  color: red;
}

Вы можете добавлять комментарии в любом месте кода:

Пример

p {
  color: red;  /* Установите цвет текста на красный */
}

Комментарии могут занимать несколько строк:

Пример

/* Это
много строчный
комментарий */

p {
  color: red;
}

HTML и CSS комментарии

Из учебника по HTML вы узнали, что вы можете добавлять комментарии к своему источнику HTML с помощью синтаксиса <!--...-->.

В следующем примере мы используем комбинацию комментариев HTML и CSS:

Пример

<!DOCTYPE html>
<html>
<head>
<style>
p {
  color: red; /* Установите цвет текста на красный */
}
</style>
</head>
<body>

<h2>Мой заголовок</h2>

<!-- Эти параграфы будут красными -->
<p>Привет Мир!</p>
<p>Этот параграф стилизован под CSS.</p>
<p>Комментарии CSS не отображаются в выходных данных.</p>

</body>
</html>
